이 문제는 map을 사용하여 풀었다.


먼저 입출력에 예시를 보면

classic 장르는 1,450회 재생되었으며, classic 노래는 다음과 같습니다.
고유 번호 3: 800회
재생고유 번호 0: 500회 재생
고유 번호 2: 150회 재생

pop 장르는 3,100회 재생되었으며, pop 노래는 다음과 같습니다.
고유 번호 4: 2,500회 재생고유 번호 1: 600회 재생

이런식으로 고유번호로 몇번을 재생했는지 알아야 하기 때문에 map을 사용하기에 적합하다 생각했다.

하지만 여기서 문제가 생긴다.

바로 map은 중복된 값을 넣을 수 없다는 점이다.

메인 코드
int main() {
	vector<int> vt;
	vt = solution({ "classic", "pop", "classic", "classic", "pop" }, { 500, 600, 150, 800, 2500 });

	for (auto item : vt)
		cout << item << "\n";
}


아쉬운점

구조체로 만들어서 했으면 더욱 쉽게 풀었을 것 같다.
